🌍 REST Countries Explorer

Frontend Mentor Challenge

A responsive front-end project built for the REST Countries API with color theme switcher challenge on Frontend Mentor.
Explore countries, search, filter by region, toggle light/dark mode, and view detailed country info.
<br />
Live Demo : [https://rest-countries-explorer-five.vercel.app/]
🚀 Features

List all countries with flag, name, population, region, and capital

Search by country name

Filter by region

Dark/Light mode toggle

Country details page: native name, population, region, subregion, capital, currencies, languages, borders

Fully responsive design

🛠️ Tech Stack

HTML5, CSS3 (Flexbox, Grid, CSS variables)

JavaScript (Fetch API, DOM manipulation, localStorage, event handling)

⚙️ How to Run

Clone the repo:

git clone https://github.com/Mohamed-Amine-Nafia/Rest-Countries-Explorer.git


Open the folder and run a local server (Live Server in VS Code recommended)

Open index.html in the browser

🧠 What I Learned

Fetching and rendering API data

DOM manipulation and event handling

Theme toggling with CSS variables

Managing page navigation with localStorage

Responsive design with CSS Grid and Flexbox
